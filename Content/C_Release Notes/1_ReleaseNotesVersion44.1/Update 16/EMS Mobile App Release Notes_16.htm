Description

Scan QR Code

QR Code Events List Was Empty After Selecting Back Button in Booking Details

When users scanned a QR code to view bookings in the events list, the events list was empty after the users selected the Back button in Booking Details.

Fix: Modified functionality so that when users scan a QR code to view bookings in the events list, the bookings display in the QR Code events list after users select the Back button in Booking Details. 

TFS item 31033

New Booking > Find a Room

Find a Room Search Results Not Showing Available Rooms

When searching for an available room to book, the Find a Room search results did not show any rooms as available even though there were rooms available.

Fix: Modified functionality so that when users view the Find a Room search results, all available rooms will display in the results list.
