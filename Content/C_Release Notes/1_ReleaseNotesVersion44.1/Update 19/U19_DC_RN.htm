LDAP Configuration

Users Could Not Assign LDAP Groups

When utilizing process templates with Oracle LDAP authentication, users were unable to assign LDAP groups to these templates. An error message was received. 

Fix: Modified functionality so that users can assign the LDAP group to the desired template.
