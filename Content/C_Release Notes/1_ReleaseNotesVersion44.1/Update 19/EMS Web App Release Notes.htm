EMS Web App Release Notes V44.1, Update 19

This section describes updates to your EMS software. 

See Also: Contacting Customer Support and Updating to the Latest Version of EMS

Name: V44.1 Update 19

Release Date: November 30, 2017

Patch: 44.1.19000.201711170

Build #:  44.1.19000.875

Fixes  

Area

Description

Everyday User Process Template

Override Default Setup/Teardown Times Were Not Displaying Correctly

Users attempting to create a reservation with an Everyday User Process Template in the EMS Desktop Client were experiencing no Setup or Tear-down time being displayed in the EMS Web App.

Fix: Modified functionality so that if the Everyday User Process Template is set to override defaults, the EMS Web App should display the setup/teardown times to accurately reflect the next available event time for a space.

TFS item 30572

Skype for Business

"Join Online Meeting" Link Was Missed in the Invitation Email After Adding Attendee 

Users who created a Skype for Business meeting and later edited the booking to add attendees were not seeing the Join Online Meeting link displayed. Attendee should have received an e-mail with the "Join Online Meeting" link. Only one email should be received. 

Fix: Modified functionality so that the Join Online Meeting link is included in the invitation and only one email is received.

TFS item 30957

Update Booking

When Updating Booking, "Cancel reason is required" Message Was Appearing

Users editing and updating a booking were experiencing a "Cancel reason is required" message. 

Fix: Modified functionality so that no error message appears and booking is updated without errors.

TFS item 31340

Room Filter

For Custom Links, Room Filter Type Displayed All Rooms Instead of the Selected One

Users creating a custom link and using the room filter type were seeing reservations with different rooms from different buildings. 

Fix: Modified functionality so that only reservations with the selected room from the selected building display.

TFS item 32805

Forgot My Password

Attempting to Get New Password Created an Error

Users who clicked the "Forgot My Password" link to get a new password were receiving the error, "There was a problem attempting to complete the action you requested. Please try again at a later time."

Fix: Modified functionality so that a new password is sent to users and an error message is not received.
