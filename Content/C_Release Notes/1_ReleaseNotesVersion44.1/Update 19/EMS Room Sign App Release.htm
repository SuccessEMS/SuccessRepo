Hardware

Aura Room Sign Hardware Integration

The EMS Room Sign App now integrates with the Aura Tablets by Resource Express. This is a turn-key hardware solution (rather than a consumer tablet), that has been designed specifically for room signs.

TFS item 33016

 

Fixes 

Area

Description

Reservation Timebar

"00:00" Appeared on Reservation Timebar

EMS Room Sign App with the browser set to English UK (with 24-hour format) was displaying "00:00" in the reservation timebar.

Fix: Modified functionality so that the reservation timebar does not display "00:00." 

TFS item 33685

Mobile Operating System

Display Size Issue on iOS 11.1.1

The EMS Room Sign App on iOS 11 was experiencing a display size issue with a gap at the bottom.  This issue appeared when the EMS Room Sign App was configured as a desktop app on Safari.  

Fix: Modified functionality so that the EMS Room Sign App displays properly on iOS 11 when configured as a desktop app on Safari.
