Parameters

Secondary Authentication Parameters/Labels Have Been Added

The parameters and labels listed below have been added to the EMS Room Sign App global settings.

Parameters:

  • Use Secondary Authentication Type for Badge Authentication: Yes/No; Default = No
  • Secondary Auth Group Identification Method
  • Secondary Auth Group Authentication Method
  • Secondary Auth Group Authentication Field

Labels:

  • Secondary Auth Button Label: Default: ‘I forgot my badge’
  • Secondary Authentication Username Label: Default: ‘External Reference’
  • Secondary Authentication Password Label: Default: 'Password'
