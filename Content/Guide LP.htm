Using EMS for Microsoft Outlook

The EMS for Micrsoft Outlook add-in proides one-click access to self-service room reservations and resource booking from within the familar Outlook personal scheduling interface. Users can find available rooms, reserve them and book resources, equipment, catering, etc. - all from within Microsoft Outlook.
