Edit Booking Not Validating Time Zone 

Reported Issue

Issue Resolution

Edit Booking > Change Event Times

When users changed event times on existing bookings, the following error message occurred: 

"The date(s) you requested violates your first allowed booking date."

Modified validations to properly check the time zone and allow the Edit Booking so the error no longer occurs.

New Bookings from Room Details Defaulting to Incorrect Start Time 

Reported Issue

Issue Resolution

Browse Locations > Select Room > Room Calendar > New Reservation

When users began a new booking from the Room Details calendar view, the event start time was incorrectly set to the first available time, when it should have been set to the next hour from "now."

Modified functionality to ensure that event start time in this situation properly defaults the next hour from "now."

PAM_AllowCriticalBookingChanges Parameter Ignored 

Reported Issue

Issue Resolution

Edit Booking 

When users attempted to make critical edits to a booking in environments that had this parameter activated, thew were unable to click the Edit button as expected.

Modified functionality to ensure that when this parameter is set, the Edit button remains available so users can make critical booking changes.
