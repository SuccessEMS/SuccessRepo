Fixes 

Area

Description

Video Conference Resource

VC Resource Service Order Not Being Created on VC Template 

For VC templates, resource tied to room was not being added upon room booking.

Fix: Modified functionality so that the service order for resource that is tied to the selected room should be created when using a properly configured VC template. 

Services

Not All Services Display in Outlook Add-In

For some templates, not all resources and categories were displaying as expected in the services tab.

Fix: Modified functionality so that all services display.
