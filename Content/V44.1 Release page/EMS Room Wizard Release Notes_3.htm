Database Authentication Using Active Directory Fails When EMSUser is Disabled

When Active Directory (AD) was used for database authentication and EMSUser was disabled on the SQL server, an error occurred.

Fix: Modified functionality so that users can disable EMSUser and continue to authenticate successfully in this manner.
