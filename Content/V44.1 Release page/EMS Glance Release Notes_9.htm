Log In

Enhance Authentication Validations  

Enhanced authentication validations so that only the group or user that owns the meeting can edit or cancel it. If a user logs in with a different group and attempt to edit or end the meeting, they will now see the message, "'No Matching clients found." Additionally, modified the Log In fields so that if the user enters an invalid group or username, the Password field is cleared when the message above presents so they can try again without needing a keyboard.
