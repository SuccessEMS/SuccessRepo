Fixes 

Area

Description

Book

Canceled Bookings Were Showing in the Book at the Building Level

User could cancel a booking that was in progress. The booking status would be set to (none) and the event would display incorrectly in the Book at the building level.

Fix: Modified functionality so that the status changed to the rule violation status defined on the template. 

Schedule View

When Meeting Time Was Changed, An Error Message Stating Room Not Available Appeared 

If a user cancelled a booking that was in progress but the process template was set to not allow cancellation of bookings in progress, the booking status would be set to (none) and the event would display incorrectly in the Book at the building level.

Fix: Modified functionality so that no error message appears and the change is made successfully.
