Past Events Not Listed After Viewing Booking Details

Reported Issue

Issue Resolution

Viewing Reservation> Booking Details > Back Button

When users opened a reservation, advanced to view a single booking's details, and then clicked the Back button to return to the reservation, past bookings were no longer listed.

Modified functionality to ensure that when users navigated to a booking details screen and then retreated to view the parent reservation, past bookings are properly listed if available.
