New Appointment

EMS Reservations Are Not Being Created Using the Save and Close Function in Outlook

When creating a New Appointment in Outlook, clicking Save and Close (instead of Save/Send) was not creating an EMS booking (room was not being booked and EMS ID was not added to the body of the Appointment).

Fix: Modified functionality so that an EMS Reservation can be created in Outlook using the Save and Close function for a New Appointment.
