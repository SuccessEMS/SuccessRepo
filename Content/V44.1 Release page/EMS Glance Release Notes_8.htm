EMS Glance Release Notes V44.1 Update 8 (EMS Glance)

This section describes updates to your EMS software.  Click on the title for each item below to expand for more details.  

Important: There is a known iOS issue where homelinks on iPads (i.e. a full screen Safari browser page that is saved as an app) do not always clear their cache even if you do so manually. This issue may cause some of your devices to have errors. If you see an error, please close Glance on the iPad and re-open it using the homelink.

Upgrading Glance Caused Error For Home Links on iPad

When users upgraded from older versions of EMS Glance on iPads, and they had set the application as a favorite on their Home screen, the link caused an error.
