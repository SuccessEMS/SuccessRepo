Name: V44.1 Update 12

Release Date: June 30, 2017

Patch: 2017.06.29.0

Build #:  44.1.1200.546

Enhancement

Area

Description


Maintain Inventory

Users Are Alerted to Remaining Inventory Quantity When Adding Resources

When adding resources to a reservation, EMS for Outlook will now alert the user to the number of available resources remaining if the quantity selected is not available.

Fix 

Area

Description


Loading Times

Outlook Plug-in Was Slow to Start and Was Being Disabled by Outlook

EMS for Outlook registry keys were modified to prevent the add-in from being disabled automatically by Outlook for some customer environments.

Fix: Modified functionality so that plug-in was not slow to start.
