EMS for Microsoft® Outlook Add-In User Guide

The EMS for Microsoft Outlook add-in provides one-click access to self-service room reservation and resource booking from within the familiar Outlook personal scheduling interface. Users can find available rooms, reserve them and book resources—such as A/V equipment, catering, etc.—all from within Microsoft Outlook.
