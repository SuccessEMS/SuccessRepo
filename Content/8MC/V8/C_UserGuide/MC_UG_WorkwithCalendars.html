Work with Calendars

When anyone, guest or user, first access EMS Master Calendar, the Default page opens. All calendars that have been defined as public are available from this page. Anyone, guest or user, can view all public events that have posted to these public calendars. If you want to view the events that are posted to the private calendars to which you have been granted access, then you must log in to EMS Master Calendar and open the Master Calendar Home page. You can view events for both single and multiple calendars. Also, a variety of options of available for working with the calendar display and the events.
