Install or Upgrade Database Components

Installing/Upgrading the Database Component in EMS Workplace / Campus / Enterprise / Legal / District 

  1. Obtain the EMSAPIScript_WebService.sql file.
  2. Open Microsoft SQL Server Management Studio.
  3. Within Microsoft SQL Server Management Studio, select File > Open > File… and locate EMSAPIScript_WebService.sql.
  4. Select your EMS database and execute the query (Query > Execute).  When query execution has completed, a ‘Command(s) completed successfully’ message will display in the Results section.
