View Today's Events in EMS Kiosk App

The Today’s Events page in EMS Kiosk lists all the events scheduled for today for rooms in the Kiosk profile.

Today's Events Page

View Map 

To view a floor map that shows the location of the scheduled event:

  1. Click the View Map icon for the event.

    Sample Floor Map for a Scheduled Event

  2. To return to the Today’s Event page after viewing a floor map, click Back, which is displayed in the upper-left corner of the Floor Map page.
