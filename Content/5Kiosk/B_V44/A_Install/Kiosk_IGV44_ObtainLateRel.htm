Obtain the Latest Release of EMS Kiosk 

The latest release of the EMS Kiosk can be downloaded from the Downloads Portal.

Log in and download EMS Kiosk (EMSKiosk.msi). Required for both first time installations and upgrades.

Launch EMS Kiosk 

  1. The first time you access EMS Kiosk, you will be redirected to the config.aspx page (http://[ServerName]/EMSKiosk/config.aspx) to enter your server and database information. Enter your EMS SQL Server name and Database. Then click Verify Server to continue.

  1. When the Kiosk Profile section appears, select a profile for this Kiosk and click Save.

Note: Kiosk Profiles are configured within the EMS Desktop Client application. Instructions can be found in the Configure EMS Kiosk Profiles.
